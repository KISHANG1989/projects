Introduction

AdminPro is a modern, responsive, and clean admin dashboard template built with Bootstrap 5. It includes a variety of custom components, charts, and pages to help you get started quickly with your next project.

Version: 1.0.0
Author: Codemitr

Installation

AdminPro is a static HTML template, so no build tools like Webpack or Gulp are required to run it in its basic form. However, a local server is recommended for loading JSON language files correctly (due to CORS policies).

To start, simply open html/index.html in your browser, or use a live server extension in VS Code.

Customization

Dark Mode

Dark mode is handled via assets/js/dark-mode.js and assets/css/dark.css. The state is saved in localStorage.

Theme Colors

You can change the primary theme color dynamically. The themes are located in assets/css/themes/. To set a default theme other than 'blue', modify assets/js/theme-switcher.js.

RTL Support

RTL is automatically enabled when the Arabic language is selected. This is handled in assets/js/lang.js by switching the Bootstrap CSS file to bootstrap.rtl.min.css and setting dir="rtl" on the HTML tag.
