# Indian University ERP System

This project is a comprehensive ERP system for Indian Universities, designed to comply with the National Education Policy (NEP) 2020.

## Modules

The system includes modules for all major departments:
- Registrar
- Exam
- Security
- Hostel
- Mess
- Academic (NEP compliant with ABC)
- Schools
- E-governance
- DSW / DSA
- Accounts
- IQAC
- HR
- Transport

## Installation

1. Ensure PHP is installed.
2. Run `php database/setup.php` to initialize the SQLite database.
3. Start the server using `php -S localhost:8000`.
